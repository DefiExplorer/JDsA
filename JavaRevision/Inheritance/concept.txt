super keyword:
It can be used to access the immediate parent class without creating an object.
This keyword is used by the subclass whenever it need to refer to its immediate
super(or parent) class.
\\
Note: - 
If constructor is overloaded in the parent class A, then in child class B
we will need to redefine the constructor before using super keyword.
